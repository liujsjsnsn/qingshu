# 情书网站 - 致我最爱的人

一个温馨的交互式情书网页，用浪漫的动画效果传达爱意。

## 功能特色

- ✨ 精美的3D翻转动画效果
- 💕 持续心动的爱心动画
- ✉️ 打字机效果展示情书内容
- 🎵 背景音乐播放
- 📱 完全响应式设计，支持移动端
- ♿ 支持键盘操作（空格键/回车键）
- 🌐 SEO优化和加载优化

## 技术栈

- HTML5 语义化结构
- CSS3 动画和响应式布局
- Vanilla JavaScript 交互效果
- 现代浏览器兼容

## 部署方式

### 快速部署（推荐）
1. **GitHub Pages**：免费托管，简单易用
2. **Vercel**：一键部署，速度极快
3. **Netlify**：功能丰富，支持自定义域名

### 手动部署
1. 将所有文件上传到服务器
2. 配置Web服务器（Nginx/Apache）
3. 确保静态文件可访问

## 文件结构

```
letter/
├── letter.html    # 主页面文件
├── letter.css     # 样式文件
├── letter.js      # 交互逻辑
├── bac.jpg        # 背景图片
├── qlx.mp3        # 背景音乐
└── README.md      # 项目说明
```

## 自定义建议

### 修改内容
- 在 `letter.js` 中修改情书文本内容
- 替换 `bac.jpg` 为自己的背景图片
- 替换 `qlx.mp3` 为其他音频文件

### 配色调整
- 修改 `letter.css` 中的背景渐变色
- 调整爱心颜色和动画效果
- 更改文字颜色和字体
